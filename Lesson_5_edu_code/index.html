color ( успадковується ): слово або код кольору;

Частіше всього використовується 16-річний код ( наприклад : #37a837 - зелений ) - ми будемо його використовувати!

Цей параметр впливає на текст, або на рядкові теги ( які повністю повторють властивість тексту )

left - вирівнює по лівому краю ( ніби за замовченням, використовуємо якщо треба перезаписати параметр, наприклад з right на left )

right - вирівнює по правому краю

justify - вирівнює текст по ширині об”єкту в якому він знаходиться ( автоматично ), змінюючи пробіли між словами
