Ця універсальна CSS-властивість дозволяє одночасно задати кілька параметрів фону, а саме визначити колір, встановити фонове зображення, задати початкове положення фонового зображення, визначити як повторюватиметься фонове зображення, встановити чи прокручуватиметься фонове зображення разом із змістом і масштабувати фонове зображення відповідно до завданих розмірів. Також значення властивості background може бути вказана градієнтна заливка фону.

background-color - визначає колір фону елемента

background-image - встановлює фонове зображення елемента або задає градієнтну заливку

background-image - встановлює фонове зображення елемента або задає градієнтну заливку

background-repeat: no-repeat; - не повторює. Бачимо зображення, яке є в нас файлі того розміру.

background-repeat: repeat-x; - повтор по горизонталі

background-repeat: repeat-y; - повтор по вертикалі

background-position - задає початкове положення фонового зображення, де перше значення ( відсотки, рх, слово ) визначає розташування по горизонталі, друге - вертикалі, 0 0; - за замовченням ( або left top ; )

background-attachment - встановлює, чи буде прокручуватися або проскролюватися зображення разом із вмістом елемента ( блоком ).

background-size - масштабує фонове зображення згідно заданим розмірам. У цифрах, де перше значення ( проценти або рх ) визначає ширину, друге - висоту. Або спеціальні значення.

Можна одночасно задати зображення і градієнт, використовуючи multi background
